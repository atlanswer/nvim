# Neovim Configs

## Prerequisites

- lua-language-server
- stylua
- cargo
- tree-sitter-cli: `cargo install --locked tree-sitter-cli`
- bun: executing tools from npm, e.g., prettier
- uv: executing Python tools, e.g., ruff
- zig
- zls
- clangd

## Usage

- `:Lazy`: Sync packages with `S`
- Relaunch Neovim and you're good to go
